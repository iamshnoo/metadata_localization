# QA Data Surfaces

This directory contains two benchmark surfaces.

## Legacy

- `qa_metacul`
- `README_qa_metacul.md`
- continent JSON files in the top-level `qa_data/` tree

These files are preserved as the original downstream benchmark used in earlier versions of the paper.

## Current

- `localnewsqa_core/`

`LocalNewsQA-Core` is the current benchmark-building surface for MAPLE. It contains the explicit/ambiguous split pipeline, Batch API request builders, evidence-retrieval scaffold, verification scaffold, and human-audit sampler.
